# Finding files

## `find` command

syntax</br>

`find <starting dir> -name <name of file>`</br>

## `locate` command

`locate <file>`</br>

## `which` command

tells you where a binary is being exicuted</br>
`which <command>`</br>

## whereis

tells you where the command is exicuting from and where the man page is</br>
syntax</br>
`whereis <command>`</br>
